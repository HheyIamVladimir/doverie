# 🔒 Доверие

Приватный мессенджер с чатами, группами, каналами и эфирами.

## ✨ Возможности

- 💬 **Личные чаты** — переписка один на один
- 👥 **Группы** — общение с несколькими людьми
- 📢 **Каналы** — публикации для подписчиков
- 📡 **Эфиры** — живые трансляции
- 📞 **Звонки** — голосовые звонки через Jitsi
- 🌗 **16 тем** — кастомизация внешнего вида
- 📵 **Оффлайн-режим** — работает без интернета (PWA)
- 📱 **Устанавливается как приложение** — на Android и iOS

## 🚀 Деплой на Render

1. Форкни репозиторий
2. Зайди на [render.com](https://render.com)
3. New → Web Service → подключи GitHub
4. Настройки:
   - **Build Command:** `npm install`
   - **Start Command:** `node server.js`
5. Deploy!

## 🛠 Технологии

- **Backend:** Node.js + Express
- **Frontend:** Vanilla JS (без фреймворков)
- **База данных:** JSON-файл (`database.json`)
- **Файлы:** multer (загрузка фото)
- **PWA:** Service Worker + Web App Manifest

## 📁 Структура

```
├── index.html          # Весь фронтенд
├── server.js           # Сервер
├── package.json        # Зависимости
├── manifest.json       # PWA манифест
├── service-worker.js   # Оффлайн-режим
├── icon-192.svg        # Иконка приложения
├── icon-512.svg        # Иконка приложения (большая)
├── database.json       # База данных (создаётся автоматически)
└── uploads/            # Загруженные фото (создаётся автоматически)
```

## ⚙️ Локальный запуск

```bash
npm install
node server.js
# Открой http://localhost:8080
```
